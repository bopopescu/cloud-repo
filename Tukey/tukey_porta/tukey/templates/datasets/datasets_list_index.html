Public Data Commons

Repository for public data sets of scientific interest, hosted on the OSDC.
The OSDC has 1PB of public data in a wide variety of disciplines.   These data sets are freely available and can be downloaded over the internet or high performance networks for analysis locally.   Information on how to download can be found at the links below for each individual dataset.   All recipients of OSDC resource allocations can also compute directly over the data in the Public Data Commons, without having to download them locally.  Datasets hosted in the OSDC Public Data Commons are reviewed periodically as part of our resource allocation process.   If you have suggestions about data that should be included, please let us know at info@occ-data.org.
